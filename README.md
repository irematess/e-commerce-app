# e-commerce-app

## Project Setup

```sh
git clone https://github.com/irematess/e-commerce-app.git
```

```sh
cd e-commerce-app
```

```sh
yarn / npm install
```

### Compile and Hot-Reload for Development

```sh
yarn dev / npm run dev
```


### Backend 

```sh
npx json-server fakedb/database.json
```
