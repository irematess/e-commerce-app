# e-commerce-app

## Project Setup

```sh
git clone https://github.com/irematess/e-commerce-app.git
```

```sh
git clone cd e-commerce-app
```

```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Compile and Minify for Production

```sh
yarn build
```



### Backend 

```sh
npx json-server fakedb/database.json
```
